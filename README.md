# rexgena

Regular Expression Generator from multiples input string without using a NLP nor Machine learning techniques.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     rexgena:
       github: dakad/rexgena
   ```

2. Run `shards install`
